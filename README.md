# RBC Signals Website Rebuild

Modern static website rebuild using Next.js 14+ (App Router), TypeScript, Tailwind CSS, and Shadcn/UI.

## Project Overview

Migrating https://rbcsignals.com from WordPress to a modern static site with automated content scraping and SEO preservation.

See [SPECIFICATION.md](./SPECIFICATION.md) for complete project specification.

## Tech Stack

- **Framework**: Next.js 16 (App Router) with static export
- **Language**: TypeScript 5+ (strict mode)
- **Styling**: Tailwind CSS v4 + Shadcn/UI
- **Content**: Hybrid MDX (pages) + JSON (structured data)
- **Hosting**: Cloudflare Pages

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
# or
task dev

# Run scraper
task scrape:homepage  # Homepage only (Phase 1)
task scrape           # Full site (Phase 2)

# Build static site
npm run build
# or
task build

# Preview built site
task preview

# Quality checks
task check  # Runs: format check, lint, type check, build
```

## Project Structure

```

├── app/                    # Next.js App Router
├── components/             # React components
│   ├── ui/                # Shadcn/UI components
│   ├── layout/            # Header, Footer, Navigation
│   ├── sections/          # Hero, CTA, Features
│   └── mdx/               # MDX custom components
├── lib/                    # Utilities
├── scripts/                # Scraper scripts
├── content/                # Generated content (MDX + JSON)
├── public/                 # Static assets
└── reports/                # Scraper reports
```
